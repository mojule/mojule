# mojule

A convenience package which packs the following into a single module:

- @mojule/dom-plugins
- @mojule/is
- @mojule/json-dom-plugins
- @mojule/json-tree
- @mojule/schema-tree
- @mojule/string-tree
- @mojule/templating
- @mojule/transform
- @mojule/tree
- @mojule/tree-factory
- @mojule/utils
- @mojule/vdom
- @mojule/vfs

## Install

`npm install mojule`

## Example

```javascript
const mojule = require( 'mojule' )

const { jsonTree, schemaTree, vdom } = mojule
```
